How it works

Synthesis

AI reads every source covering a story and writes a living briefing that evolves as new information emerges. Every sentence links to its original source.

Claim Tracking

When officials say one thing and evidence shows another, Dornt surfaces the contradiction. Claims are tracked across sources with clear status badges.

Coverage Lens

See how different outlets frame the same event. Dornt identifies distinct narrative frames and shows which sources fall on each side.

Storylines

Individual events don't happen in isolation. Dornt detects when stories are part of a larger narrative arc, connecting cause and effect across weeks.

Community Reactions

See how different Reddit communities respond to the same story — sentiment analysis, key perspectives, and highlight quotes from real discussions.

Daily Briefing

The top stories, synthesized and delivered daily. A finished intelligence product — not just a list of links.

About Dornt

Dornt is an open-source news intelligence platform. It runs entirely on GitHub Actions — every day, the pipeline ingests fresh content, clusters and analyzes it with AI, and deploys a fully static site. No servers, no accounts, no tracking.

Inspired by Grond.ai via Max Frazer. Built with Claude Code, Node.js, and OpenRouter. View on GitHub
